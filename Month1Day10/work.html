作用域

Javascript的作用域只有两种：全局作用域和本地作用域，本地作用域是按照函数来区分的。 var a = 1;
var f = function () {
console.log(a);
var a = 2;
console.log(a);
}
f();
console.log(a);

js在语法分析阶段会扫描本作用域内的所有变量声明并提前，于是 f 函数体内就被提前，结果依次是：undefined,2,1。

易错题 var someuser = {
name: 'byvoid',
func: function() {
console.log(this.name);
} };
var foo = {
name: 'foobar'
};
someuser.func(); // 输出 byvoid
foo.func = someuser.func;
foo.func(); // 输出 foobar
name = 'global';
func = someuser.func;
func(); // 输出 global

函数作用域的嵌套关系是定义时决定的,而不是调用时决定的,也就 是说,JavaScript 的作用域是静态作用域,又叫词法作用域,这是因为作用域的嵌套关系可以在语法分析时确定,而不必等到运行时确定。
