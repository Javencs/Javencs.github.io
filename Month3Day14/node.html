1.什么是nodeJS

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。 Node.js 使用了一个事件驱动、非阻塞式 I/O 的模型，使其轻量又高效。 Node.js 的包管理器 npm，是全球最大的开源库生态系统。

安装完nodejs，就可以打开命令窗口进行操作了

键入node进入命令交互模式，可以输入一条代码语句后立即执行并显示结果

查询使用的版本:node -v

脚本模式:console.log("Hello World");

常用的Linux命令

改变目录：cd 文件夹名

查看当前目录：pwd

常看当前目录下的详细信息：ls -l(显示最近一次修改时间)

创建目录：mkdir 文件夹名

创建文件：touch

删除文件：rm 文件名

删除文件夹：rm -r 文件夹名

复制文件：cp 文件名 复制后的文件名

复制文件夹和文件夹中的内容：cp -r

>移动文件：mv

查找命令：find
