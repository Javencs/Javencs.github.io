在js中，有一个Error对象，我们通常可以用它来设置错误信息，或者设置异常捕捉。

Error 对象也可以用语法显式创建，或用 throw 语句抛掉。在两种情况下，都可以添加选择的任何属性，以拓展 Error 对象的能力。

var newErrorObj = new Error();
var newErrorObj = new Error(number);
var newErrorObj = new Error(
number, //number:与错误相联的数字值。如果省略则为零。
description //description是描述错误的简短字符串。如果省略则为空字符串。
);
