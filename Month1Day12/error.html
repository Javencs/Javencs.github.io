js Error对象

Event对象

在js中，有一个Error对象，我们通常可以用它来设置错误信息，或者设置异常捕捉。

Error 对象也可以用语法显式创建，或用 throw 语句抛掉。在两种情况下，都可以添加选择的任何属性，以拓展 Error 对象的能力。

var newErrorObj = new Error();
var newErrorObj = new Error(number);
var newErrorObj = new Error(
number, //number:与错误相联的数字值。如果省略则为零。
description //description是描述错误的简短字符串。如果省略则为空字符串。
); 抛出Error throw new Error(0,"Error Demo");
new Error可省略:
throw("Error Demo");
捕获Error try catch finally语句: try{
..可能错误的语句..}
catch(e){
..错误发生后的处理..}
finally{ // finally不是必须的
..完成后执行的语句块..}

window.onerror错误捕获,各个浏览器对onerror的支持不同，IN发生error时，正常代码会执行；在火狐中，代码将结束；Safari只支持onerror事件处理。

onerror事件会传给回调函数3个默认参数:Msg: 错误信息；Url: 发生错误的文件的Url；Num: 错误发生位置的行号。

img 应用onerror：img src="sample.jpg" onerror="javascript:alert('图像载入错误');" ；如果图像无法加载，将弹窗“图像载入错误”

处理Error 判断错误类型:
catch(e){
if(e.name=="RangeError")
alert("错误提示");}
或
catch(e){
if(e instanceof TypeError)
alert("错误提示");}
